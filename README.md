# 

## Overview
This is a React application structured for maintainability and ease of development. It includes organized folders for components, utilities, and styles.

## create react app 
 1. npx create-react-app my-app
 2. npm install
 3. npm run dev
 4. git init
 5. git add .
 6. git commmit -m "git add"
 7. git remote add origin
 8. git branch -M main
 9. git push -u origin main
     

## Folders

- `components/` → All UI components, including the main `App.jsx`.
- `utils/` → Utility functions, helpers, or API calls.
- `styles/` → Global or component-specific CSS files.

---

## Scripts
- `npm start` → Run development server
- `npm build` → Build production version
- `npm test` → Run tests

## Notes

- `dependencies` → Required at runtime (e.g., react, react-icons)
- `devDependencies` → Only for development (e.g., eslint)
